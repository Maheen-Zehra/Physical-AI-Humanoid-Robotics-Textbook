# Introduction to NVIDIA Isaac
---
---

NVIDIA Isaac is a comprehensive robotics platform that combines hardware and software to accelerate the development and deployment of AI-powered robots. It leverages NVIDIA's expertise in AI, accelerated computing, and simulation to provide a complete solution for robotics development.

## Platform Components

The NVIDIA Isaac platform consists of:

- **Isaac ROS**: Accelerated perception and navigation libraries
- **Isaac Sim**: High-fidelity simulation environment
- **Isaac Lab**: Open-source framework for robot learning
- **Jetson Platform**: Edge computing hardware for robotics

## Isaac ROS

Isaac ROS provides hardware-accelerated perception and navigation capabilities with GPU-accelerated algorithms, CUDA-optimized libraries, and seamless ROS 2 compatibility. It includes pre-built packages for common robotic functions.

## Isaac Sim

Isaac Sim is a high-fidelity simulation environment built on NVIDIA Omniverse. It features the PhysX physics engine for accurate simulation, RTX rendering for photorealistic graphics, and synthetic data generation capabilities for training AI models.
